## Usage
1\. Choose a pre-trained model in *'trained_model'* folder.

2\. Modify paths for selected model and testing dataset in *'run.py'* file.

3\. Run **BMVOS**!!
```
python run.py
```

4\. If necessary, download pre-computed results at [here](https://drive.google.com/file/d/1AokjFfA8Gb65dIQ3T3V4lM5CuOfKuIqm/view?usp=sharing).
